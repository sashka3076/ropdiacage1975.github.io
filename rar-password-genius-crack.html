      ZIP is an archive format used to store files in the compressed state. It is popularly used in Windows operating systems. After downloading a file in the ZIP format, the contents of the file requires unpacking before opening and editing them. However, as compared to the ZIP format, RAR has more advanced features like the following: more convenient multivolume archives, tight compression, multimedia text modes, AES-128 protection, helping to repair an archive even in case of physical data damage, and Unicode support to process non-English file names.
